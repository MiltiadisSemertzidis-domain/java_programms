# Variables

This folder contains Java programs related to variables and data types.

## Java Primitive Data Types

| Type   | Size    | Description                  |
|--------|---------|------------------------------|
| byte   | 1 byte  | Stores small integers        |
| short  | 2 bytes | Stores medium integers       |
| int    | 4 bytes | Stores standard integers     |
| long   | 8 bytes | Stores very large integers   |
| float  | 4 bytes | Stores decimal numbers       |
| double | 8 bytes | Stores large decimal numbers |
| char   | 2 bytes | Stores a single character    |
| boolean| 1 bit   | Stores true or false         |
